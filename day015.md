## 力扣每日一题:
  230.二叉搜索树中第K小的元素
  [https://leetcode.cn/problems/kth-smallest-element-in-a-bst?envType=study-plan-v2&envId=top-100-liked]

### 思路：
- 中序遍历得到排序数组，然后直接看第K小的元素就行，不过这个不能实现对频繁修改的搜索树进行查找，得用平衡二叉搜索树？吓哭了，不过在修改搜索树的时候同步修改数组应该也是一种方法。
  ```python
  class Solution(object):
    def kthSmallest(self, root, k):
        """
        :type root: Optional[TreeNode]
        :type k: int
        :rtype: int
        """
        ans = []
        def dfs(root, ans):
            if root is None:
                return None
            dfs(root.left, ans)
            ans.append(root.val)
            dfs(root.right, ans)
        
        dfs(root, ans)
        return ans[k - 1]
